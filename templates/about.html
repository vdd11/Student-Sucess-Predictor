About This Project

This project uses the UCI Student Performance dataset to build a machine learning classifier that predicts whether a student is likely to pass based on academic and school-related indicators.

The goal is not to replace educators. The goal is to show how data science can support early intervention, tutoring decisions, and student success analytics.

Built with Python, Pandas, Scikit-Learn, Flask, HTML, CSS, and Render deployment settings.
